## ğŸ“ Bit-Level Insight (for range understanding)

- A signed `n`-bit type uses 1 bit for the sign.
- Remaining `n-1` bits represent the magnitude:
  - Max value = \(2^{n-1} - 1\)
  - Min value = \(-2^{n-1}\)

**Example**: 
- 16-bit signed int:
  - 1 sign bit + 15 bits for value
  - Range = `-32768 to 32767`
